# Library Management System (Java)

This project is a simple console-based Java application created as self-practice to strengthen fundamental Java programming skills.

The program simulates a basic library system where users can manage a list of books through a text-based menu.

## Features
- Add books to the library  
- Display all stored books  
- Search for a book (case-insensitive)  
- Remove a book with user confirmation  
- Basic input validation  

## Concepts Practiced
- Java fundamentals  
- Loops and conditional statements  
- ArrayList usage  
- User input handling using Scanner  
- String manipulation
